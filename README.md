# Student Management System

A command-line based Student Management System developed in Python for managing student records efficiently. The application provides complete CRUD (Create, Read, Update, Delete) functionality with persistent data storage using JSON.

## Features

* Add Student Records
* View All Students
* Search Students by ID
* Update Student Information
* Delete Student Records
* Persistent Data Storage using JSON
* Duplicate Student ID Prevention
* Interactive Menu-Driven Interface

## Technologies Used

* Python
* JSON
* File Handling

## Core Functionalities

### Add Student

Creates a new student record and stores it in the system.

### View Students

Displays all available student records.

### Search Student

Retrieves student information using the student ID.

### Update Student

Modifies existing student details.

### Delete Student

Removes a student record from the system.

### Persistent Storage

Stores data in JSON format, ensuring records remain available across application restarts.
